基本环境配置

简单记录一下自己的日常学习娱乐环境

这篇文章可以配合另一篇文章 (opens new window)使用。目前对于安卓刷机比较麻烦，像我的平板打算刷lineageos，结果没人适配，只能作罢。路由器打算刷openwrt，不过这台华为路由肯定不行，等换小米吧。

# 桌面配置

# Kde

效果可以看一下我以前某乎回答后面一段,可以看一下效果视频 (opens new window)，我个人平时不太喜欢开动态桌面的，现在kde也不用了，换成了i3wm。

全局使用Nord (opens new window)主题，我现在桌面顶栏和latte-dock都没有放，所有工作都是靠krun搜索，壁纸也是静态壁纸，但是这些东西还是都配置上了，为的是哪天装逼时候使用

kwin插件：forceblur (opens new window)，gridtiling (opens new window)

kde顶栏插件：Better inline clock (opens new window)，Weather Widget (opens new window)，mcOS BC inline Bettery (opens new window)，Resource Monitor(fork) (opens new window)

latte-dock：mcOS Monterey (opens new window)

动态壁纸：使用SmartER Video Wallpaper (opens new window)，需要安装依赖

sudo pacman -S gst-libav gst-plugins-bad gst-plugins-base gst-plugins-good gst-plugins-ugly

# i3wm

我个人参考了三份配置：ayamir/i3-dotfiles (opens new window)，dikiaap/dotfiles (opens new window)，denisse-dev/dotfiles (opens new window)，配置放在仓库 (opens new window)里。我个人不喜欢花哨的效果，如果不是因为看视频画面会撕裂，我自己连picom都不会开，基本外出就会把picom关掉，开视频啥的再打开。

i3wm网上中文资料很少，要搜linux桌面那基本会出来大量的ubuntu+gnome，或者manjaro+kde这种，自然的，DE的确适合新手，我也不建议一上来就搞wm，等用到wm的时候，也基本用不到什么教程了，基本都是看着别人配置改，然后去依赖仓库看文档。

# gnome

我当时在某乎写了个回答 (opens new window)，不过现在帐号注销了，半年不用gnome，里面那些插件也不知道现在更不更新。

# 浏览器

我个人使用的是firefox和brave浏览器，原版chrome用不惯，firefox个人用的是第三方UI：minimal-functional-fox (opens new window)，代理工具使用clash (opens new window)和tor (opens new window)。

Startpage：Bento (opens new window)

插件：Midnight Lizard (opens new window)，SwitchyOmega (opens new window)，Tampermonkey (opens new window)，Simple Translate (opens new window)

推荐油猴脚本：搜索引擎优化 (opens new window)

# 开发工具

我个人主要使用neovim进行开发，vscode以前是主力，自从配好nvim以后就没怎么用。vim和kate偶尔会用，kate的话加lsp插件实验了一下还是不能作为ide使用，权当一个文本编辑器吧。Kate主题个人配置了nord。Qtcreator当时写qt用过，算是电脑上唯一的专用IDE。

# vscode

个人使用主题：glassit-linux (opens new window)，nord (opens new window)，外加vim插件，配置可以在我的仓库 (opens new window)找到。

# VIM

目前使用neovim+neovide (opens new window)，配置都在仓库里，参考了lvim作者的教程 (opens new window)，他配套有个视频，b站有搬运 (opens new window)，不过后面部作者没出视频，自己对照插件文档基本也能改。vim我自己作为备选项，配置同样放在仓库里，但是没有代码补全功能，仅作为一个备用文本编辑器使用，配置参考vimplus (opens new window)。

至于说vim难用这个问题，我一开始也这么觉得，后来学习了之后把所有编辑器都装上fake vim，关于快捷键难记的问题，其实不然，可以参考一下这篇文章 (opens new window)，同问题下的高赞写的也不错，还有一篇相对系统的文章 (opens new window)，基本你花点时间把这三篇看完就没啥问题，vim我个人感觉，一个无法替代的是宏录制功能，fakevim我自己没见过有支持的。

如果说你确实不想自己配置，我自己用过两个发行版，spacevim (opens new window)，和lvim (opens new window)，还有同类项目nvchd (opens new window)，不过我个人没有用过。lvim默认不带代码运行功能，我个人是使用了sniprun (opens new window)和code runner (opens new window)。我以前是用的spacevim (opens new window)，同样是一个非常优秀的发行版，配置更加轻松，但lvim好处在于可定制性更强，而且效率更高。

# 终端

笔者使用zsh作为默认终端，zsh插件安装教程都可以在插件项目主页找到，所以不建议使用ohmyzsh，会降低效率，插件：zsh-syntax-highlighting (opens new window)，zsh-autosuggestions (opens new window)，zshrc可在在仓库 (opens new window)找到。

终端主题：p10k (opens new window)，注意按照教程配置字体，否则无法正常显示

我自己常用终端是alacritty (opens new window)，konsole自带主题nord，alacritty配置可以在仓库 (opens new window)找到。

我目前把很多终端程序都换成了rust替代品，可以看一下我的zshrc，本人rust粉，但你要我自己写那就不型了，目前还在学习，打算把cpp，rust，python当作三门主力语言。neofetch主题用的neofetch-theme (opens new window)

fish的话作为一个备选，prompt用的starship (opens new window)。

# 阅读写作

# 阅读：

笔者使用zathura (opens new window)作为默认阅读器，okular (opens new window)作为备用阅读器，后者自己把color mode调整成nord配色。zathurarc可以在仓库 (opens new window)找到。

推荐配合wudao词典 (opens new window)，set selection-clipboard clipboard设置了默认选中进行复制，而wudao是比较方便的命令行词典，可以直接复制进终端进行查询

# 写作：

markdown：使用typora (opens new window)+picgo (opens new window)，具体可以参考typora (opens new window)官方文档，主题使用nord (opens new window)。

word，ppt：的使用libreoffice (opens new window)全家桶

latex：neovim+lsp。

# 其他使用的软件和配置

# 输入法：

个人使用fcitx5 (opens new window)+四叶草 (opens new window)和双拼 (opens new window)作为输入法，主题使用fcitx5-nord (opens new window)。

# bt下载：

使用deluge (opens new window)和qbittorrent (opens new window)，使用tracker：trackerslist (opens new window)。

# 图像影音：

图像处理使用GIMP (opens new window)，看图使用ristretto (opens new window) ，影像处理使用shotcut (opens new window)，视频播放器使用VLC (opens new window)，屏幕录制使用OBS (opens new window)。

# 网盘：

使用阿里云盘+webdav (opens new window)，好处是免费，不限速，容量大，如果手头有带宽合适服务器更加方便，我服务器带宽不行，就本地运行了。

# RSS：

使用自建RSShub (opens new window)+feed43 (opens new window)进行同步，阅读器使用easyrss (opens new window)+feeder (opens new window)+newsflash (opens new window)，feed43可以参考官方教程 (opens new window)。如果手头没有服务器自建RSShub可以参考这篇教程 (opens new window)。

本站rss见底部链接，或者使用github官方rss (opens new window)。(注：head标签里面rss不能用，因为目前还没有配置)

# 社交娱乐：

疼讯软件：qq使用Icalingua++ (opens new window)，微信我个人电脑上没有安装，aur里面wechat-uos (opens new window)看起来比较火，腾讯会议 (opens new window)官方支持linux。我平时不用qq,weixin，这个基本也就是传文件时候用，平时tg/lark/matrix。

音乐：个人使用网易第三方：yesplaymusic (opens new window)。和NeteaseCloudMusicGtk4 (opens new window)。

游戏：steam (opens new window)官方支持linux，支持的游戏可以在protondb (opens new window)上找，proton我个人尝试郭GTA5，除了我个人设备没独立显卡无法流畅运行外没啥3毛病，像csgo，我的世界等原生支持linux的游戏自然没啥问题。

# 其他：

虚拟机使用kvm+qemu+Quickemu (opens new window)/virt-manager (opens new window)，和virtualbox (opens new window)，

安卓链接使用scrcpy (opens new window)，虚拟机使用genymotion (opens new window)。

启动盘刻录使用balenaetcher (opens new window)，修复使用gparted (opens new window)。

远程控制使用rustdesk (opens new window)

# 安卓配置

# 阅读：

阅读：ReadEra (opens new window)，词典：欧陆词典 (opens new window)，推荐这两个原因是可以比较好的适配，而且没有广告。

# 终端：

使用termux (opens new window)，我用这个连ssh写代码。实验了下proot-distro (opens new window)，配上vnc理论上可以开启图形界面，但我实验以下效果并不理想。使用纯tty界面用gcc编译helloworld用了1.5秒，而且rust用起来会有问题，所以还是别搞这东西了，老老实实用ssh吧

# 设备互联：

使用kdeconnect (opens new window)，不得不说KDE家软件相当不错，Kate，dolphin等都是极品。

# root工具：

使用通用的magisk (opens new window)，不得不说安卓这块比电脑端麻烦的多。
